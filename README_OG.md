## Work flow is as follows.

1) Make sure you are on YourBranch.
2) 'git pull origin master'
3) Work
4) 'git add ...'
5) 'git commit ...'
6) 'git push origin YourBranch'
7) 'git checkout master'
8) 'git merge YourBranch'
9) 'git push origin master'
